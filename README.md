# Bikers.web  バイク乗り向けの情報共有サイト

Herokuを用いて公開してます、ページへは[こちら](https://biker-app.herokuapp.com/)

##機能
* 写真投稿、表示
* 地図投稿、表示
* コメント機能
* ログイン機能
* 検索機能

## 環境
* Ruby version 2.4.9
* rails 5.2.2
* bundelr 1.3.0

## 追加したGem
* bootstrap 4.4.1
* jquery-rails 4.3.5
* geocoder 1.6.1
* carrierwave 1.3.1
* cloudinary 1.11.1
* dotenv-rails 2.7.5
* kaminari 1.2.0
* annotate 3.1.0
